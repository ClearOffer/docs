Use the Compensation tab to define base pay and any variable components. Fields marked required must be completed to save.

## Base pay

<CodeGroup>
```text Annual Salary
Enter the annual amount. We format the number as you type.
```

```text Currency
Choose the pay currency (USD, EUR, GBP, etc.).
```

```text Payment Frequency
Weekly, Biweekly, Twice a Month, or Monthly.
```
</CodeGroup>

<Tip>
Use market‑appropriate numbers. You can update values anytime before sending.
</Tip>

## Equity (optional)

<Tabs>
<Tab title="RSUs">
- Set Shares and Current Share Price  
- Estimated value = Shares × Current Share Price
</Tab>

<Tab title="Stock Options">
- Set Shares, Strike Price, and Preferred Price  
- Estimated value = Shares × (Preferred − Strike)
</Tab>
</Tabs>

<Info>
The “Estimated Equity Value” is calculated for guidance. It isn’t a guarantee of future value.
</Info>
